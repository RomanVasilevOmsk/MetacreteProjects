Все узлы ремонта:

Узлы ремонта ж\б колонн:

  • C оголенной поверхностью прокорродировавшей арматуры (без усиления арматуры)
  • C коррозионной трещиной
  • C оголением и разрывом арматуры (хомутов)
  • C установкой дополнительной арматуры

Узлы ремонта ж\б поверхностей и элементов:

  • Фасадов сооружений
  • Слоем до 20 мм
  • Сборных бетонных элементов
  • При выполнении ремонтных работ в интервале темпертур от –10 °С до +5 °С

Узлы ремонта ж\б плит:

  • С трещинами вдоль арматурных стержней (ширина раскрытия до 3 мм), а также с оголенной поверхностью прокорродировавшей арматуры
  • С заменой промасленного защитного слоя бетона
  • С восстановлением нижней грани

Узлы ремонта прочих поверхностей:

  • Бетона с разрушением сечения плиты до 100% с оголением арматуры
  • Сквозных трещин по кирпичной кладке (ширина раскрытия до 10 мм)
  • Потолочной поверхности перекрытия слоем до 50 мм
  • Плиты пола
  • Узел установки болтов (анкеров) в существующий фундамент

Узлы ремонта ж\б балок:

  • Продольного и поперечного перекрытия
  • С оголениями и разрывом арматуры (хомутов)
  • Боковой грани, разрушенной на глубину до 100 мм, с оголением и коррозией арматурных стержней
  • Проход через плиту монолитного ж/б балочного перекрытия (вырубленный бетон с разрывом арматуры)
